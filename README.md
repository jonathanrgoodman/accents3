# Evidence that cultural groups differ in their abilities to detect fake accents: a follow up

The raw data can be found <a href="https://github.com/jonathanrgoodman/accents3/blob/main/accent-data-3-raw-data.csv">here</a>; the  .rmd for all initial analyses are <a href="https://github.com/jonathanrgoodman/accents3/blob/main/accents-3-analyses.Rmd">here</a> (view the html version <a href="https://htmlpreview.github.io/?https://github.com/jonathanrgoodman/accents3/blob/main/accents-3-analyses.nb.html">here</a>).
